# AQMAR Arabic Wikipedia NER

A mBERT example for Arabic NER. 

Sota from [Liu et al., (2019)](https://www.aclweb.org/anthology/W19-4607.pdf)

### Running
``python arabic_ner.py``

### Results

    Model               Test F1
    Liu et al., (2019) 75.82
    stanza              74.3
    mBERT               73.9
    XLM-R               69.3
    GigaBERTv3          76.59
